Introduction


Unity is a development environment for programming games of all kinds. From powerful computer games to simple cell phone games,
everything is possible.
In Unity, 3D - 2D games or animated films can be produced.
Since unity was originally developed for 3d games, it is often called unity3d, which is not entirely correct.
In this course we will cover the most important steps from download to publication.


Download

To download Unity you should first download Unity Hub because there you can easily download and view the different versions of Unity. Unity Hub can be downloaded here.


Create the first project

In order to create our first project we first open the Unity hub. If you have already created a project and you open Unity hub for the first time, click add and search for your project. In our case we don't have that and that's why we press new. With the arrow directly to the right of new you can change the version if you have more than one.

Now you can choose which type of you project should be the most important are 2d and 3d.

2D: a regular 2d game

3D: a regular 3d game

3D whith Extras: a 3d game with post processing

High-Definition Rp: 3d that uses the HD Render Pipeline

Lightweight RP: 3d that uses the Lightweight Render Pipeline

VR Lightweight RP: 3d with Lightweight VR Render Pipeline



Finally, we have to choose a name and the location.
