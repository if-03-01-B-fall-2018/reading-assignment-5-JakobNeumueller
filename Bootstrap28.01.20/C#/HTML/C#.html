C# - Some first tipps for newcomers

In this course you will learn the basics of C# programming, what you need to start already today and what you will be able to do once you have finished it.
C# is a general-purpose and object-oriented (class-based). It was developed around 2000 by Microsoft as part of its .NET initiative, and later approved as an international standard ISO. C# is one of the programming languages designed for the Common Language Infrastructure (CLI).
